# Vim-Configuration

### .vimrc for Competitive Programming: [.vimrc](https://github.com/aam-icpmst/Vim-Configuration/blob/main/.vimrc)
**`Usages:`**
1. Open Linux Terminal in `/home` directory
2. Type `vim .vimrc`
3. Press `Enter`
4. Paste the `.vimrc` file
5. Press `Esc`
6. Type `:wq`
7. Press `Enter`

That's it!
